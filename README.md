A *phun* Balatro mod themed around **PHIGHTING!**.

> #### IMPORTANT NOTE!
> This mod requires [Steamodded](https://github.com/Steamodded/smods) to be set-up properly!
>
> Please take the time to go through the installation guide on Steamodded's github page!


As of version *v1.0*, the mod currently adds:

![image](https://file.garden/aFSSiTCQJTZAsdGN/balatro/pppreleases/ppp_v1-0-0.png)

These jokers should all be compatible with [Talisman](https://github.com/SpectralPack/Talisman/releases/latest) (Making it compatible with the Cryptid mod) **however** I can't guarantee that there won't be oddities!

## Bugs / Issues?
Feel free to go to the [Issues Tab](https://github.com/yuuworks/Phighting-Joker-Pack/issues) to report them! I'll try my best to fix any bugs as soon as I become aware of them!

## Installation
Please refer to the Steamodded guide on how to install the mod!
You can simply download the [source code from the latest release](https://github.com/yuuworks/Phighting-Joker-Pack/releases/latest) and treat that as the mod's folder!
